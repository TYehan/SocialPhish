## ⭐SocialPhish - Remastered [![GitHub stars](https://img.shields.io/github/stars/TYehan/SocialPhish.svg?style=social)](https://github.com/TYehan/SocialPhish) 
***Remastered version of <b>SocialPhish</b> by <a href="https://github.com/xHak9x">xHak9x</a>***
- 32 templates +1 customizable.
- Standalone self-hosting.
- Local tunnel hosting or ngrok.
- Beginner Friendly.

---
### ⭐Requirements:
Unix-like Operating System (Linux, ParrotOS, KaliLinux, ..etc) or compatible environment on Windows (WSL, Cygwin).

---

### ⭐Tutorial Demo Video:

https://github.com/TYehan/SocialPhish/assets/85949077/7ee5a92e-3338-4ed3-adac-d4eb576ffa06

---
### ⭐Usage:

```
git clone https://github.com/TYehan/SocialPhish.git
cd SocialPhish
chmod +x socialphish.sh
./socialphish.sh
```
---
#### Note:
If this error occurs: `zsh: ./socialphish.sh: bad interpreter: /bin/bash^M: no such file or directory`
- Run the following command to fix it:
  
```
sed -i 's/\r$//' socialphish.sh
chmod +x socialphish.sh
./socialphish.sh
```
